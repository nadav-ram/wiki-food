# wiki-food

#### Installation:

1) Clone from GitHub

```bash
git clone https://github.com/nadav-ram/wiki-food.git
```

2) Install requirements

```bash
pip install -r requirements.txt
```

3) Run app

```bash
on windows:
cd flaskapp/ && py app.py
on unix:
cd flaskapp/ && python3 app.py
```

4) Try out

```bash
open your browser to localhost:5000
```
